## 앤트로픽, "값싼 에이전트"를 노린 Claude Sonnet 5 공개

앤트로픽이 6월 30일(현지시간) 중형 모델 라인업의 새 버전인 **Claude Sonnet 5**(모델 ID `claude-sonnet-5`)를 공개했습니다. 핵심 메시지는 "불과 몇 달 전이면 더 크고 비싼 모델이 필요했던 자율 작업을, 이제 Sonnet급 가격에 돌릴 수 있다"는 것입니다.

### 자율 실행에 초점 맞춘 '가장 에이전트다운 Sonnet'

Sonnet 5는 앤트로픽이 "역대 가장 에이전트다운 Sonnet"이라고 표현한 모델로, 스스로 계획을 세우고 브라우저·터미널 같은 도구를 사용하며 사람의 개입 없이 다단계 작업을 끝까지 수행하도록 설계됐습니다. 컨텍스트 창은 **100만 토큰**을 지원하고, 출시와 동시에 claude.ai의 무료·Pro 사용자 기본 모델로 지정됐습니다. 앤트로픽은 "브라우저와 터미널 같은 도구를 사용해 자율적으로 실행할 수 있다"며, 이전 버전 대비 환각과 아첨(sycophancy) 발생률도 낮췄다고 밝혔습니다. [TechCrunch](https://techcrunch.com/2026/06/30/anthropic-launches-claude-sonnet-5-as-a-cheaper-way-to-run-agents/) / [Anthropic System Card](https://www.anthropic.com/claude-sonnet-5-system-card)

### 벤치마크·가격: Opus 4.8에 근접, 비용은 절반 수준

성능 지표에서 Sonnet 5는 에이전트 코딩 벤치마크 **SWE-bench Pro에서 63.2%**를 기록해 이전 Sonnet 4.6(58.1%)을 크게 앞섰고, 최상위 모델 Opus 4.8(69.2%)과의 격차를 좁혔습니다. 컴퓨터 조작 벤치마크 **OSWorld-Verified에서는 81.2%**로 Sonnet 4.6(78.5%)을 넘어섰고, 일부 지식노동 벤치마크에서는 Opus 4.8을 소폭 앞서기도 했습니다. 가격은 **8월 31일까지 100만 토큰당 입력 2달러·출력 10달러**의 도입가로 제공되며 이후 입력 3달러·출력 15달러로 조정됩니다. 출시 시점부터 Claude API, Claude Code, Cursor, VS Code, GitHub Copilot에서 곧바로 사용할 수 있습니다.
