## 🏭 Anthropic 'Claude Design' 발표에 Figma 7% 급락 — 모델 기업이 인접 SaaS를 직접 위협하다

파운데이션 모델 기업이 모델 API 판매를 넘어 애플리케이션 레이어로 직접 내려오자, 해당 시장의 상장 SaaS 기업 주가가 즉각 반응했습니다. AI 경쟁의 전선이 '모델 성능'에서 '누구의 시장을 먹느냐'로 이동하고 있음을 보여준 사건입니다.

### Anthropic, 'Claude Design' 출시 — Figma 주가 7.28% 하락·시총 12억 달러 증발 (4월 17일)

2026년 4월 17일 Anthropic이 텍스트 프롬프트로 프로토타입·와이어프레임·슬라이드·디자인 시스템을 생성하는 **Claude Design**을 출시했습니다. Claude Opus 4.7 기반으로, GitHub 코드베이스·디자인 파일을 업로드하면 팀의 색상·타이포그래피·컴포넌트를 자동 적용한 디자인 시스템을 만들고, 이를 Claude Code로 넘겨 실제 배포까지 연결합니다. 시장 반응은 즉각적이었습니다. **Figma 주가는 장중 최대 7.28% 하락해 $18.84로 마감(직전 종가 $20.32)**, 단일 거래일에 **시가총액 약 12억 달러**가 증발했고, **Adobe 주가도 약 1.5% 하락**했습니다. 공교롭게 출시 3일 전 Anthropic 최고제품책임자(CPO) Mike Krieger가 Figma 이사회에서 사임해 의도를 둘러싼 해석을 낳았습니다. 왜 중요한가: 모델 기업이 디자인 툴이라는 **애플리케이션 레이어로 직접 진입**해 기존 SaaS의 해자를 잠식하는 신호로, 디자인·문서·코딩 등 'AI로 대체 가능한 워크플로우'를 가진 상장 기업들의 밸류에이션 리스크가 현실화되고 있음을 보여줍니다.
