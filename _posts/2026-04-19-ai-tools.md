## 🛠️ General Compute ASIC 추론 클라우드·MAI-Image-2-Efficient — 에이전트 전용 인프라와 이미지 AI의 속도 혁신

4월 14~18일 AI 인프라와 이미지 생성 도구 두 영역에서 기존 GPU 중심 패러다임을 탈피한 새로운 접근이 등장했습니다. General Compute는 에이전트 워크로드에 특화된 ASIC 기반 추론 클라우드를, Microsoft는 기존 이미지 생성보다 22% 빠르고 4배 효율적인 새 모델 변형을 선보였습니다.

### General Compute: ASIC 우선 AI 에이전트 추론 클라우드 — 5월 15일 GA, 프리필·디코드 독립 스케일링, 수력 전력 데이터센터

2026년 4월 18일 General Compute Inc.가 자율 AI 에이전트를 위한 ASIC 우선 추론 클라우드 플랫폼을 발표하며, 5월 15일 일반 공개를 예고했습니다. 현재 early-partner 운영 단계입니다. 핵심 기술 특징: (1) **GPU 대신 목적 특화 ASIC** — 범용 그래픽 처리 장치 대신 AI 추론에 특화된 맞춤 가속기를 사용합니다. (2) **프리필·디코드 분리 스케일링** — LLM 추론의 두 단계(컨텍스트 처리와 토큰 생성)를 워크로드에 따라 독립적으로 스케일링해 자원 효율을 높입니다. (3) **수력 전력·에어쿨링** — GPU 클러스터 대비 낮은 전력 밀도로 운영 비용을 절감합니다. 에이전트 중심 설계: 오픈소스 LLM 다중 패밀리·파라미터 크기 지원, 사용자 자체 모델 배포 가능, 업계 표준 API로 기존 앱 최소 변경 통합. CTO 제이슨 굿이슨: '지난 20년은 개발자를 위해 구축했다. 앞으로 20년은 에이전트를 위해 구축할 것이다.' [Winger Daily](https://www.wingerdaily.com/2026/04/18/general-compute-launches-asic-first-inference-cloud-for-autonomous-ai-agents/) / [OpenPR](https://www.openpr.com/news/4478116/general-compute-launches-asic-first-inference-cloud)

### Microsoft MAI-Image-2-Efficient: 기존 대비 22% 빠르고 4배 효율적인 이미지 생성 모델 — 경쟁 모델 대비 평균 40% 빠름, 입력 $5/백만 토큰

2026년 4월 14일 Microsoft가 MAI-Image-2의 효율화 변형인 MAI-Image-2-Efficient를 Azure AI Foundry에서 공개했습니다. MAI-Image-2는 Arena.ai 이미지 모델 리더보드 3위에 데뷔한 모델로, 이번 Efficient 버전은 동일 아키텍처에서 속도와 효율성을 극적으로 향상했습니다. 핵심 성능: 레이턴시·GPU 사용량 기준으로 MAI-Image-2 대비 **22% 빠르고 4배 효율적**이며, 주요 경쟁 이미지 생성 모델 대비 평균 **40% 빠른** 생성 속도를 기록합니다. 가격: 텍스트 입력 **$5/백만 토큰**, 이미지 출력 **$19.50/백만 토큰**. 적합한 사용 사례: 고화질 사실감보다 처리량이 중요한 고볼륨 워크플로우(대규모 콘텐츠 파이프라인, 실시간 이미지 생성 API 등). 원본 MAI-Image-2는 사실감 극대화가 필요한 경우에 계속 권장됩니다. [Microsoft TechCommunity](https://techcommunity.microsoft.com/blog/azure-ai-foundry-blog/introducing-mai-image-2-efficient-faster-more-efficient-image-generation/4510918)
