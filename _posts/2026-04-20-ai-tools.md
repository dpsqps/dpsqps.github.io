## 🛠️ OpenAI Codex 데스크톱, '백그라운드 컴퓨터 유즈'로 맥 전체를 조작하는 에이전트가 되다

코딩 보조에 머물던 Codex가 데스크톱 전반을 스스로 조작하는 범용 에이전트로 외연을 넓혔습니다. 'AI가 코드를 써주는' 단계에서 'AI가 내 컴퓨터를 직접 쓰는' 단계로 넘어간 변화입니다.

### OpenAI, Codex 데스크톱 앱에 '백그라운드 컴퓨터 유즈' 추가 — 맥의 모든 앱을 보고·클릭·입력 (4월 16일)

2026년 4월 16일 OpenAI가 Codex 데스크톱 앱에 '백그라운드 컴퓨터 유즈(background computer use)'를 도입했습니다. 이제 Codex는 자체 커서로 화면을 보고(see)·클릭하고(click)·입력하며(type) 컴퓨터의 모든 앱을 사용할 수 있고, **여러 에이전트가 사용자의 작업을 방해하지 않으면서 맥에서 병렬로** 작동합니다. 이번 업데이트는 매주 Codex를 사용하는 **300만 명 이상의 개발자**를 겨냥했으며, **90개 이상의 신규 플러그인**, 인앱 브라우저, 영구 메모리(persistent memory), 며칠~수주 뒤에 작업을 재개하는 스케줄링 레이어, SSH를 통한 원격 devbox 연결, PR 리뷰·다중 파일/터미널 보기 등 개발 워크플로우 기능이 함께 추가됐습니다. 왜 중요한가: 이번 변화로 Codex의 포지셔닝이 'multi-agent 코딩 앱'에서 **범용 데스크톱 워크플로우 도구**로 이동했습니다. 코딩 에이전트 경쟁이 'IDE 안의 자동완성'을 넘어 'OS 레벨에서 사람을 대신해 일하는 워커'로 무대를 옮긴 셈입니다.
