## Claude API 소개

Anthropic Claude API는 강력한 LLM을 자체 서비스·업무 자동화에 통합할 수 있는 인터페이스다. Python, TypeScript 공식 SDK를 제공하며, 메시지·비전·스트리밍·함수 호출(Tool Use) 등 다양한 기능을 지원한다. 안전성과 명확한 답변이 강점으로, 기업 고객의 채택률이 빠르게 늘고 있다.

## 기본 사용법

```python
import anthropic
client = anthropic.Anthropic(api_key="YOUR_API_KEY")
message = client.messages.create(
    model="claude-3-5-sonnet-20241022",
    max_tokens=1024,
    messages=[{"role": "user", "content": "한국 AI 산업 동향을 요약해줘"}]
)
print(message.content[0].text)
```

System Prompt로 페르소나·출력 형식·제약 조건을 설정하면 일관된 자동화 파이프라인을 구성할 수 있다.

## 실전 자동화 시나리오

**콘텐츠 자동 생성:** RSS 피드→Claude→블로그 포스트 자동 작성. **문서 분류·요약:** 이메일·보고서를 Claude로 분류하고 핵심만 추출. **코드 리뷰 봇:** PR 생성 시 GitHub Action으로 Claude API 호출해 자동 코드 리뷰 댓글 게시. **고객 지원 자동화:** CRM 티켓을 Claude로 분석해 우선순위와 초안 답변을 생성.

## Tool Use(함수 호출) 활용

Tool Use 기능으로 Claude가 외부 API·데이터베이스를 직접 호출하는 에이전트를 구성할 수 있다. 날씨 조회, 주가 검색, 캘린더 조작 등의 도구를 정의하면 Claude가 사용자 요청에 따라 적절한 도구를 선택해 실행한다. 2026년에는 멀티에이전트 오케스트레이션 API가 강화될 예정이다.
