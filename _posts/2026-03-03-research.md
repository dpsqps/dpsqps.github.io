## AI 연구 & 논문 동향 (2026.03.03)

### SleepLM: 수면과 언어를 연결하는 파운데이션 모델

arXiv에 공개된 **SleepLM**은 수면 다원검사(polysomnography) 데이터와 자연어를 연결하는 멀티모달 파운데이션 모델입니다. 10,000명 이상, 100,000시간 이상의 수면 데이터와 텍스트를 결합한 최초의 대규모 수면-텍스트 데이터셋을 구축하고, 대조 정렬·캡션 생성·신호 재구성을 통합한 학습 목표를 제안했습니다. 수면 단계 분류, 이상 이벤트 탐지 등에서 제로샷·퓨샷 성능이 기존 최고 모델을 능가했습니다. [arXiv](https://arxiv.org/abs/2602.23605)

### HumanMCP: MCP 도구 검색 평가 데이터셋

**HumanMCP** 논문은 Model Context Protocol(MCP) 서버 환경에서 LLM의 도구 검색 능력을 평가하는 최초의 대규모 데이터셋을 제안합니다. 308개 MCP 서버의 2,800개 도구에 대해 다양한 사용자 페르소나와 쿼리를 생성하여, 실제 사용자가 어떻게 요청하는지를 반영한 현실적인 벤치마크를 구축했습니다. [arXiv](https://arxiv.org/abs/2602.23367)

### KAIST '뇌를 닮은 AI': 예측 오류 시 재사고

KAIST 연구진이 예측이 틀렸을 때 자동으로 한 번 더 생각하는 신경과학 기반 AI 모델을 개발했습니다. 인간 뇌의 예측 오류 처리 메커니즘을 AI에 적용한 연구로, 단순 피드포워드 방식의 한계를 극복하고 더 안정적인 추론 능력을 보여줬습니다. [AI타임스](https://www.aitimes.com/news/articleView.html?idxno=207400)

### 구글·유튜브 STATIC: 추천 성능 1000배 향상

구글 딥마인드와 유튜브 연구진이 LLM 기반 생성 추천 시스템에서 다양한 제약 조건을 빠르고 효율적으로 처리하는 **STATIC(Sparse Transition Matrix-Accelerated Trie Indexing for Constrained generation)** 프레임워크를 발표했습니다. 기존 임베딩 기반 검색 대비 성능을 최대 1000배 향상시켰다고 밝혔습니다. [AI타임스](https://www.aitimes.com/news/articleView.html?idxno=207435)

### AML 컴플라이언스: LLM 에이전트로 자동화

**금융 범죄 방지(AML) 컴플라이언스** 영역에서 LLM과 RAG를 결합한 에이전틱 시스템이 제안됐습니다. AI 에이전트가 웹을 검색하고 관련 문서를 수집·분석하여 각 인물의 부정적 미디어 지수(AMI)를 자동 산출하는 방식으로, 기존 키워드 검색 방식의 높은 오탐률 문제를 해결했습니다. [arXiv](https://arxiv.org/abs/2602.23373)
