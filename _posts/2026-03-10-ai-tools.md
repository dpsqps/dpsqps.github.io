# AI 도구 동향: 코드 리뷰, 로컬 에이전트, 지속성 메모리의 시대

## 앤트로픽 Claude Code Review: AI 생성 코드 검증 도구

[앤트로픽이 Claude Code에 코드 리뷰 기능을 추가](https://techcrunch.com/2026/03/09/anthropic-launches-code-review-tool-to-check-flood-of-ai-generated-code/)했습니다. 멀티 에이전트 시스템이 AI 생성 코드를 자동으로 분석하여 로직 오류, 보안 취약점, 코드 품질 문제를 식별합니다. '바이브 코딩' 트렌드로 AI 생성 코드의 양이 폭발적으로 증가하는 상황에서, 기업 개발자들이 코드 품질을 유지할 수 있는 핵심 안전망으로 주목받고 있습니다. 엔터프라이즈 고객을 겨냥한 이 기능은 PR(풀 리퀘스트) 단계에서 자동 검토를 제공하며, 기존 코드 리뷰 도구 대비 더 깊은 맥락 이해를 바탕으로 동작합니다.

## 리퀴드 AI LocalCowork: 완전 오프라인 AI 에이전트

소형 모델 전문 기업 리퀴드 AI가 [LFM2-24B-A2B 모델과 LocalCowork 데스크톱 에이전트](https://www.aitimes.com/news/articleView.html?idxno=207724)를 공개했습니다. 클라우드 API 없이 노트북에서 AI 모델, 데이터, 도구 실행을 모두 처리합니다. 금융기관이나 의료 환경처럼 데이터를 외부 서버로 전송할 수 없는 규제 산업에서의 수요를 정확히 겨냥한 솔루션입니다. MIT 라이선스로 오픈소스 공개되어 개발자 커뮤니티의 참여도 적극 유도하고 있습니다.

## 구글 Always-On Memory Agent: 지속성 메모리 오픈소스화

구글이 [Always-On Memory Agent](https://www.aitimes.com/news/articleView.html?idxno=207715)를 MIT 라이선스로 공개했습니다. 기존 AI 에이전트의 한계였던 '세션 종료 후 기억 소멸' 문제를 해결하는 이 시스템은 벡터 DB 대신 자연어 기반 지속성 메모리를 활용합니다. 에이전트가 장기적으로 사용자의 선호와 맥락을 기억하고 누적 학습할 수 있어, 개인화 AI 어시스턴트 구현의 새로운 가능성을 열었습니다. 구글 클라우드 공식 GitHub에 MIT 라이선스로 공개됐습니다.

## OpenAI의 Promptfoo 인수: AI 보안 강화

[OpenAI가 AI 에이전트 보안 스타트업 Promptfoo를 인수](https://techcrunch.com/2026/03/09/openai-acquires-promptfoo-to-secure-its-ai-agents/)한 것은 프론티어 랩들이 에이전트 보안을 얼마나 중요하게 여기는지를 단적으로 보여줍니다. Promptfoo는 프롬프트 인젝션, 탈옥(jailbreak), 데이터 유출 등 에이전트 관련 보안 위협을 탐지·방어하는 도구를 제공해왔습니다. 기업 운영에서 AI 에이전트를 안전하게 사용하기 위한 보안 레이어 강화가 업계 전체의 화두로 떠오르고 있습니다.

오늘의 AI 도구 트렌드는 한마디로 **'AI 활용의 안전화와 실용화'**입니다. 단순히 기능을 늘리는 방향에서, 실제 기업 환경에서 신뢰할 수 있게 활용하기 위한 도구들이 주목받고 있습니다. 보안, 오프라인 운영, 장기 기억 등 엔터프라이즈 환경의 실질적 요구를 해결하는 도구들이 시장을 이끌고 있습니다.
