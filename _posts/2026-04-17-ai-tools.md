## 🛠️ OpenAI Agents SDK 전면 업그레이드 — 샌드박스·서브에이전트·100개+ LLM으로 엔터프라이즈 에이전트 표준화

4월 15일 OpenAI가 발표한 Agents SDK 'Next Evolution'은 AI 에이전트 배포의 실험 단계가 끝났음을 선언합니다. 안전성, 확장성, 공급자 중립성을 동시에 강화한 이번 업데이트는 기업 AI 인프라의 새 기준을 제시합니다.

### OpenAI Agents SDK Next Evolution: 네이티브 샌드박스·서브에이전트·100개+ LLM 지원 + Assistants API 2026년 중반 종료

4월 15일 OpenAI가 Agents SDK 대규모 업데이트를 발표했습니다. 핵심 4가지 신기능: ① **네이티브 샌드박스 실행** — 에이전트가 파일·도구·코드를 격리된 워크스페이스에서만 접근하며 시스템 전체 무결성을 보호합니다. ② **구성 가능 메모리(configurable memory)** — 에이전트가 장기 실행 세션에서 컨텍스트를 유지하며 내장 스냅샷(built-in snapshotting)으로 실행 중단·재개(durable agent runs)가 가능합니다. ③ **서브에이전트(subagents)** — 오케스트레이터가 전문화된 하위 에이전트를 생성해 병렬·모듈식 태스크 분해를 처리합니다(Python·TypeScript 양쪽 추가 예정). ④ **공급자 중립성** — 100개 이상의 LLM을 표준화된 인터페이스로 지원합니다. 중요한 변화: OpenAI가 Assistants API를 2026년 중반 종료(deprecated) 한다고 발표했습니다. 모든 에이전트 워크플로우를 새 SDK로 이전하라는 공식 요청입니다. Python 우선 출시, TypeScript는 추후 제공.
