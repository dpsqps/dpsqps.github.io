## 🤖 Microsoft MAI 3종·Claude Opus 4.6 Chatbot Arena 1위 — AI 자체 모델 전쟁과 인간 선호도

4월 첫째 주, Microsoft가 OpenAI 의존에서 독립하는 자체 모델 3종을 출시하며 AI 모델 시장 경쟁을 재정의했고, Anthropic의 Claude Opus 4.6이 인간 선호도 지표 최정상을 달성했다.

### Microsoft MAI-Transcribe-1·MAI-Voice-1·MAI-Image-2: Foundry 3종 동시 출시 (4월 2일)

4월 2일 Microsoft가 자체 개발 AI 모델 3종을 Azure Foundry와 MAI Playground에 공개했다. OpenAI 파트너십과 별개로 독립적인 AI 역량을 확보하겠다는 전략적 선언이다. **MAI-Transcribe-1**은 25개 언어 지원 음성인식 모델로, 기존 Azure Fast 대비 배치 전사 속도 2.5배, GPU 비용 50% 절감을 달성했으며 가격은 오디오 1시간당 $0.36이다. **MAI-Voice-1**은 단일 GPU에서 60초 분량 고품질 음성을 1초 이내 생성하며, 수 초 분량의 음성 샘플만으로 커스텀 보이스를 생성할 수 있다. **MAI-Image-2**는 텍스트-이미지 생성 모델로 Arena.ai 이미지 모델 리더보드에서 3위를 기록했다. 세 모델이 각각 Whisper(음성인식)·ElevenLabs(음성합성)·DALL-E(이미지)와 정면으로 경쟁하는 구도다. OpenAI가 Microsoft에 독점 기술 접근권을 제공하던 구조에서 두 회사가 AI 모델 시장에서 직접 경쟁하는 단계로 전환됐음을 의미한다. [TechCrunch](https://techcrunch.com/2026/04/02/microsoft-takes-on-ai-rivals-with-three-new-foundational-models/) / [VentureBeat](https://venturebeat.com/technology/microsoft-launches-3-new-ai-models-in-direct-shot-at-openai-and-google)

### Claude Opus 4.6 LMSYS Chatbot Arena Elo 1504 — 코딩 1549, 환각 오류율 4배 개선

4월 6~7일 Claude Opus 4.6 Thinking이 LMSYS Chatbot Arena 전체 부문에서 Elo 1504로 1위를 달성했다. Anthropic이 1·2위를 독점했고, 코딩 특화 리더보드에서도 상위 5위 모두가 Anthropic 모델로 채워졌으며 Claude Opus 4.6은 코딩 점수 1549를 기록했다. Chatbot Arena는 블라인드 인간 선호도 평가를 집계하는 플랫폼으로, MMLU·HumanEval과 달리 점수 조작이 구조적으로 불가능한 가장 신뢰도 높은 모델 비교 지표다. Claude Opus 4.6 Thinking은 숨겨진 Chain-of-Thought 추론 단계를 통해 응답 전에 자체 출력을 디버깅하며, 이 구조가 기술 작업의 환각 발생률을 전 세대(4.5) 대비 4배 개선했다고 Anthropic은 주장한다. GPT-5.4·Gemini 3.1 Pro를 제치고 인간 선호도 1위를 달성한 것은 벤치마크 점수가 아닌 실제 사용자 만족도에서 우위를 입증한 것이어서, 기업 채택 결정에 직접적인 영향을 미친다. [AI World](https://aiworld.eu/story/claude-opus-46-leader-of-the-leaderboards) / [LMSYS Chatbot Arena](https://arena.ai/leaderboard/text/)
