## 🛠️ Canva AI 2.0 에이전트화·Factory AI 유니콘 등극 — 크리에이티브·코딩 도구의 에이전트 전환 본격화

4월 16~17일 디자인과 코딩 두 영역에서 AI 도구가 '보조 도구'에서 '에이전트형 시스템'으로 전환하는 두 사례가 발표됐습니다. 265M 사용자 플랫폼과 엔터프라이즈 코딩 에이전트 스타트업이 동시에 새로운 단계를 선언했습니다.

### Canva AI 2.0: 265M 사용자 플랫폼의 에이전트 전환 — 도구 체이닝·영구 메모리·Gmail·Slack·Zoom 연동, 100만 사용자 리서치 프리뷰 배포

2026년 4월 16일 Canva가 AI 2.0을 리서치 프리뷰로 출시하며 100만 명 우선 사용자에게 배포를 시작했습니다. 전 세계 265M 사용자 기반 디자인 플랫폼을 에이전트형 크리에이티브 시스템으로 전환하는 대규모 업데이트입니다. 핵심 기능 3가지: (1) **Agentic Orchestration(도구 체이닝)** — Canva의 전체 디자인 엔진에 AI가 직접 접근해, 목표를 설명하면 레이아웃 생성·포맷 조정·내보내기 준비까지 단계별로 적합한 도구를 자율 선택·실행합니다. 사용자가 단계마다 수동 조정할 필요가 없습니다. (2) **영구 메모리(Persistent Memory)** — 사용자의 작업 방식을 학습하고 브랜드 에셋 업데이트 시 관련 디자인을 자동 갱신합니다. (3) **외부 서비스 연동** — Gmail·Slack·Zoom에 연결해 이메일 대화·회의 메모에서 콘텐츠를 직접 생성합니다. Fortune: 'Canva는 조용히 세계에서 가장 많이 사용되는 AI 서비스 중 하나가 됐다.' ### Factory AI: Droids 기반 엔터프라이즈 코딩 에이전트로 $1.5B 유니콘 등극 — $150M 조달, Khosla+Sequoia+Blackstone, NVIDIA·Adobe·EY 수십만 개발자 활용

2026년 4월 16일 2023년 창업 스타트업 Factory가 Khosla Ventures 주도·Sequoia Capital·Insight Partners·Blackstone 참여로 $1억 5천만(약 2천억 원) 투자를 $1.5B(약 2조 원) 밸류에이션에 마감하며 유니콘에 진입했습니다. Factory의 핵심 제품 'Droids'는 단순 코드 생성에 그치지 않고 테스팅·코드 리뷰·문서화·배포까지 소프트웨어 개발 전 라이프사이클을 자율적으로 처리하는 에이전트입니다. 멀티 LLM 전략: Anthropic Claude·DeepSeek 등 다양한 파운데이션 모델 간 유연하게 전환해 특정 LLM에 종속되지 않습니다. 현재 NVIDIA·Adobe·EY·Palo Alto Networks·Adyen·MongoDB·Bayer·Zapier에서 수십만 명의 개발자가 Droids를 매일 활용합니다. GitHub Copilot이 코드 자동완성을 메인스트림화했다면, Factory Droids는 PR 제출부터 배포까지 '자율 실행'을 메인스트림화하는 다음 단계를 노린다.
