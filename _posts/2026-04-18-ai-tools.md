## 🛠️ OpenAI Codex 컴퓨터 사용·90개+ 플러그인, Google Gemini 개인화 이미지 — AI가 OS 레벨로 진입

4월 16~17일 OpenAI와 Google이 각자의 AI 도구를 대폭 업그레이드했습니다. AI가 이제 단순 API 호출을 넘어 사용자의 운영체제와 개인 사진 라이브러리를 직접 다루는 'OS 레벨 에이전트'로 진화하고 있습니다.

### OpenAI Codex 'for (almost) everything': macOS 앱 직접 제어·90개+ 플러그인·메모리·GPT-Image-1.5

4월 16~17일 OpenAI가 Codex 데스크탑 앱에 대규모 업데이트를 적용했습니다. 핵심 4가지 신기능: ① **컴퓨터 사용(Computer Use)** — 자체 커서로 macOS 앱 전체를 직접 보고·클릭·타이핑하며 제어합니다. 여러 에이전트가 사용자 작업을 방해하지 않고 병렬로 실행됩니다(macOS 우선, EU·UK 추후). ② **90개+ 플러그인** — Atlassian Rovo(JIRA), CircleCI, CodeRabbit, GitLab Issues, Microsoft Suite, Neon by Databricks, Render 등 MCP 서버 통합을 포함합니다. ③ **메모리(Memory Preview)** — 이전 세션의 선호·교정 사항을 기억해 반복 설명 없이 작업합니다. ④ **GPT-Image-1.5 이미지 생성** — 코드·스크린샷과 결합한 프로덕트 목업·게임 에셋 생성이 가능합니다. Anthropic의 Claude Code와 맞붙는 포지셔닝으로, OpenAI가 에이전트 개발 환경을 OS 레벨에서 직접 장악하는 전략이 구체화됐습니다. [OpenAI](https://openai.com/index/codex-for-almost-everything/) / [VentureBeat](https://venturebeat.com/technology/openai-drastically-updates-codex-desktop-app-to-use-all-other-apps-on-your-computer-generate-images-preview-webpages) / [gHacks](https://www.ghacks.net/2026/04/17/openai-updates-codex-with-computer-use-in-app-browser-memory-and-90-plus-new-plugins/)

### Google Gemini + Nano Banana 2: 내 Google Photos 기반 개인화 이미지 — 간단한 프롬프트로 가족 얼굴 반영

4월 16일 Google이 Gemini 앱의 Personal Intelligence 기능에 Nano Banana 2 기반 개인화 이미지 생성을 추가했습니다. 사용자가 Google Photos의 인물·반려동물 레이블을 Gemini와 연결하면, "나와 가족이 즐겨 찾는 곳을 클레이메이션으로 그려줘" 같은 단순 프롬프트 한 줄로 실제 얼굴이 반영된 이미지를 자동 생성합니다. 기존 AI 이미지 생성과 달리 긴 묘사 없이도 개인 맥락이 자동으로 반영되는 것이 핵심입니다. 미국 내 Plus·Pro·Ultra 구독자에게 우선 배포되며, Chrome 데스크탑 및 추가 국가로 확대 예정입니다. Google은 프라이버시 보호를 위해 "개인 사진 라이브러리로 모델을 직접 훈련하지 않으며, 특정 프롬프트와 모델 응답 등 제한된 정보만 학습에 활용한다"고 밝혔습니다. [Google Blog](https://blog.google/innovation-and-ai/products/gemini-app/personal-intelligence-nano-banana/) / [TechCrunch](https://techcrunch.com/2026/04/16/google-adds-nano-banana-powered-image-generation-to-geminis-personal-intelligence/) / [Engadget](https://www.engadget.com/ai/gemini-can-now-draw-on-your-google-data-to-personalize-the-images-it-generates-160000269.html)
