## 🛠️ 값은 바닥으로, 기기는 앱 없이 — AI 도구의 두 갈래 파괴

오늘 AI 도구 영역의 화두는 '비용'과 '폼팩터'입니다. 한쪽에서는 딥시크가 LLM API 가격을 다시 한 번 끌어내려 경쟁을 재점화했고, 다른 한쪽에서는 오픈AI가 앱 대신 AI 에이전트로 작동하는 스마트폰을 준비 중이라는 보도가 나왔습니다.

### 딥시크, API 가격 90% 추가 인하로 글로벌 LLM 최저가 경신

딥시크(DeepSeek)가 4월 24일 출시한 V4 모델 라인업에 이어, 4월 27일 보도 기준으로 '입력 캐시 히트(input cache hit)' 가격을 90% 인하하며 글로벌 LLM API 서비스의 새로운 최저가 기준을 세웠습니다. V4-Pro는 75% 할인 적용 후 입력 캐시 미스가 100만 토큰당 0.435달러, 캐시 히트는 0.0037달러, 출력은 0.87달러 수준입니다. 이는 출력 토큰 기준으로 GPT-5.5의 34분의 1, Claude Opus 4.7의 17분의 1에 불과한 가격으로, 출시 다음 날 V4-Pro API 호출량은 전일 대비 297% 급증했습니다. 이 공격적 인하는 단순한 마케팅 이벤트가 아니라 AI 에이전트 운영 비용 구조 자체를 흔드는 변수입니다. 에이전트는 한 작업에 수십~수백 번의 토큰 호출을 반복하기 때문에, 토큰 단가가 수십 배 싸지면 그동안 비용 때문에 미뤄졌던 상시(always-on) 에이전트 워크로드가 한꺼번에 경제성을 갖게 됩니다. 미국 빅테크 대비 수십 배 저렴한 가격은 개발자·스타트업 유치 경쟁의 핵심 무기로, 글로벌 LLM API 시장의 가격 전쟁을 다시 점화시켰습니다. [DIGITIMES](https://www.digitimes.com/news/a20260427PD220/deepseek-llm-api-cost-price.html)

### 밍치궈 "오픈AI, 앱 대신 AI 에이전트로 작동하는 스마트폰 개발 중"

유명 애널리스트 밍치궈(Ming-Chi Kuo)가 오픈AI가 미디어텍·퀄컴·럭스셰어와 협력해 전통적 앱 대신 AI 에이전트가 작업을 수행하는 스마트폰을 개발 중이라고 X에 공개했다고 보도됐습니다. 이 기기는 온디바이스 모델과 클라우드 모델을 결합해 다양한 요청을 처리하며, 오픈AI는 파트너들과 맞춤형 스마트폰 칩 개발도 함께 추진하는 것으로 전해졌습니다. 부품 사양은 2027년 말 확정, 양산은 2028년 시작 예정이며, 협력사로는 미디어텍·퀄컴·럭스셰어가 거론됐습니다(애널리스트 보도로 비공식이며, 오픈AI는 공식 입장을 내지 않았습니다). 이 구상이 중요한 이유는 '앱 아이콘을 누른다'는 모바일의 기본 상호작용 자체를 부정하기 때문입니다. 앱 스토어를 거치지 않고 AI 에이전트가 사용자의 의도를 받아 직접 작업을 수행하는 방식은, 애플·구글이 쥔 앱 생태계와 수수료 구조를 우회하는 '에이전트가 곧 OS'인 새로운 모바일 패러다임을 시사합니다. 오픈AI의 하드웨어 진출 움직임(2026년 하반기 첫 제품 예고)과 맞물리며 스마트폰 시장 구도 변화 가능성을 던진 보도입니다. [TechCrunch](https://techcrunch.com/2026/04/27/openai-could-be-making-a-phone-with-ai-agents-replacing-apps/)
