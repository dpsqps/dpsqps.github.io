## Microsoft 멀티모델 Critique·Anthropic 보안 사건 — 모델 경쟁의 새 국면

멀티모델 협업 전략과 AI 기업의 보안 이슈가 동시에 부각됐습니다.

### Microsoft Copilot Cowork: GPT+Claude 듀얼 모델 전략

Microsoft가 Copilot Cowork를 확대하며 'Critique' 기능을 발표했습니다. GPT가 초안을 작성하면 Claude가 정확성·인용 품질을 검토하거나 그 반대로 동작합니다. 'Council' 모드는 여러 모델의 답변을 나란히 비교합니다. 단일 모델 의존에서 멀티모델 협업으로의 전환을 보여줍니다. [Digital Trends](https://www.digitaltrends.com/computing/microsofts-copilot-cowork-arrives-with-smarter-ai-research-tools-to-spot-gaps-in-your-work/)

### Anthropic 2차 보안 사건: Claude Code 소스코드 유출

Anthropic이 Claude Code 소스코드 유출이라는 2차 보안 사건을 겪었습니다. Mythos/Capybara 유출에 이어 발생한 사건으로 보안 체계에 대한 우려가 커지고 있으나, 투명한 공개와 신속한 대응으로 처리하고 있습니다. [Crescendo AI](https://www.crescendo.ai/news/latest-ai-news-and-updates)

### Bluesky Attie: AI 기반 소셜 피드 빌더

Bluesky가 AI 기반 맞춤 피드 알고리즘 빌더 Attie를 출시했습니다. 코딩 없이 개인화된 콘텐츠 큐레이션 알고리즘을 구축할 수 있어, AI 모델의 소셜 미디어 적용 사례로 주목됩니다. [labla.org](https://www.labla.org/latest-ai-model-releases-past-24-hours/ai-releases-roundup-what-actually-launched-on-march-29-30-and-31-2026/)
