## 🛠️ Cursor 3·OpenClaw·Salesforce Slackbot — 에이전트 오케스트레이션 주류화

이번 주 AI 도구 시장은 '에이전트에게 작업을 위임하는' 패러다임으로의 전환이 코드 에디터부터 기업 메신저까지 전방위에서 동시에 선언됐다.

### Cursor 3: IDE에서 에이전트 오케스트레이션 플랫폼으로 전면 재설계 (4월 2일)

Cursor가 4월 2일 Cursor 3를 출시하며 인터페이스를 처음부터 다시 구축했다. 핵심 변화는 에디터 중심에서 에이전트 관리 플랫폼으로의 전환으로, 데스크톱·웹·모바일·Slack·GitHub·Linear에서 시작된 모든 로컬·클라우드 에이전트가 통합 사이드바에 한눈에 표시된다. 멀티 리포지터리 레이아웃과 로컬-클라우드 에이전트 간 seamless 핸드오프를 지원하며, 기존 Cursor IDE로의 전환도 언제든 가능하다. Claude Code, OpenAI Codex와의 정면 경쟁을 선언한 이 출시는 AI 코딩 도구 시장의 패러다임이 '코드 자동완성'에서 '에이전트 위임'으로 이동했음을 코드 에디터 레벨에서 공식화했다. [Cursor 공식 블로그](https://cursor.com/blog/cursor-3) / [SiliconAngle](https://siliconangle.com/2026/04/02/cursor-refreshes-vibe-coding-platform-focus-ai-agents/)

### OpenClaw: 이틀 만에 GitHub 스타 10만, 현재 30만 2천 — 로컬 에이전트 폭발 성장

오픈소스 로컬 에이전트 도구 OpenClaw(구 Clawdbot)가 이번 주 폭발적인 성장을 기록했다. GitHub 스타 10만 개를 단 2일 만에 돌파했으며 현재 누적 30만 2천 개에 달한다. 서버 없이 로컬에서 셸 명령·파일 관리·웹 자동화를 수행하며, WhatsApp·Telegram·Signal 메신저와의 통합도 지원한다. Anthropic이 4월 4일부터 서드파티 도구에 대한 구독 사용을 차단하면서, 특정 AI 서비스에 의존하지 않는 완전 로컬·독립형 에이전트에 대한 수요가 역설적으로 더욱 자극되고 있다. ### Salesforce Slackbot: MCP 통합 30개 신기능 — 47M 사용자 업무에 에이전트 내장

Salesforce가 Slackbot에 30가지 AI 기능을 추가하고 자율 업무 보조 AI로 재포지셔닝했다. Model Context Protocol(MCP)을 통해 외부 도구 연동을 지원하며, 재사용 가능한 AI 스킬 기능을 탑재해 반복적인 업무 흐름을 자동화할 수 있다. Slack의 월간 활성 사용자 수는 약 4,700만 명으로, 이 규모의 플랫폼에 에이전트 기능이 기본 내장되면서 별도 AI 구독 없이도 일상 업무에서 에이전트를 활용할 수 있는 기준선이 한 단계 높아졌다.
