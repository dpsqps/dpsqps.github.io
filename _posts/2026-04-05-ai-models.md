## 🤖 DeepSeek V4·GPT-5.4 Thinking·Veo 3.1 Lite — 오픈 가중치와 비용 압박의 교차점

어제 Gemma 4의 Apache 2.0 공개에 이어, 이번 주 모델 시장은 오픈소스 공세와 특화 모델 출시가 연속으로 이어지며 폐쇄형 프론티어의 경쟁 우위가 본격적으로 시험받고 있다.

### DeepSeek V4: 훈련 비용 $5.2M, SWE-bench 80% 돌파 — 오픈 가중치 공개

DeepSeek이 이번 주 공개한 V4는 1조 파라미터 MoE 구조이지만, 추론 시 활성화되는 파라미터는 ~370억에 불과해 실제 처리 효율이 대형 밀집 모델 대비 압도적으로 높다. 전체 훈련 비용이 약 520만 달러($5.2M)로 추산되는데, 미국 동급 모델의 훈련 비용($100M+)의 5%에 불과하다. 성능은 SWE-bench Verified 80% 초과, HumanEval 94.7%로 최상위 클로즈드 모델과 직접 경쟁하며, API 가격은 입력 토큰 100만 개당 $0.30으로 경쟁사 대비 10분의 1 이하다. Apache 2.0 오픈 가중치로 공개해 상업적 파인튜닝과 재배포가 완전 자유롭다. [NxCode](https://www.nxcode.io/resources/news/deepseek-v4-release-specs-benchmarks-2026)

### GPT-5.4 Thinking: OSWorld-Verified 75%, OS 에이전트 자율 실행

OpenAI의 GPT-5.4 Thinking은 포스트 트레이닝 스케일링(test-time compute) 방식으로 OS 레벨 에이전트 실행에 특화된 모델이다. OSWorld-Verified 벤치마크에서 75.0%를 기록해 GPT-5.2(47.3%) 대비 27.7퍼센트포인트의 도약을 이루었다. 이 벤치마크는 실제 운영체제 환경에서 복잡한 멀티스텝 작업을 자율 수행하는 능력을 측정하며, GPT-5.4 Thinking의 결과는 AI 에이전트가 실제 컴퓨터를 자율 조작하는 시나리오에 한 발 더 가까워졌음을 의미한다. [devFlokers](https://www.devflokers.com/blog/ai-news-last-24-hours-april-2026-model-releases-breakthroughs)

### Google Veo 3.1 Lite: 표준 Veo 대비 절반 이하 가격의 동영상 생성 모델

Google이 4월 초 출시한 Veo 3.1 Lite는 기존 Veo 3.1의 절반 이하 비용으로 동영상을 생성하는 경량 버전이다. Google AI Studio와 Vertex AI에서 접근 가능하며, 고퀄리티 영상보다 빠른 프로토타이핑이나 대량 생성이 필요한 기업 워크플로에 적합하다. 동영상 AI의 비용 장벽이 낮아지면서 마케팅·교육·미디어 분야의 기업급 도입 속도가 빨라질 전망이다.
