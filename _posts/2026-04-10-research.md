## 📄 Self-Distilled RLVR·In-Place TTT·RAGEN-2 — 에이전트 추론의 효율·안정성·붕괴 방지

이번 주 arXiv에서 멀티모달 벤치마크 4.69% 개선, LLM 동적 적응, 에이전트 추론 붕괴 탐지 등 에이전트 추론의 실용적 성능 향상에 집중한 논문들이 높은 관심을 받았다.

### Self-Distilled RLVR: 멀티모달 벤치마크 4.69% 향상 — 토큰 레벨 크레딧 할당으로 RLVR 안정화

4월 8일 arXiv에 등재된 Self-Distilled RLVR(Chenxu Yang, Chuanyu Qin, Qingyi Si 외)은 기존 강화학습(RLVR) 기반 포스트 트레이닝의 핵심 문제인 크레딧 할당 불안정성을 해결한다. 이 프레임워크는 모델이 자기 자신의 이전 출력을 '소프트 레이블' 교사 신호로 활용하는 자기 증류 메커니즘을 도입해, 시퀀스 레벨이 아닌 토큰 레벨에서 크레딧을 할당한다. 멀티모달 벤치마크 전반에서 베이스 LLM 대비 평균 4.69%의 정확도 향상을 달성했으며, 사전 수집된 외부 데이터 없이 자체 생성 데이터만으로 학습 안정성을 유지한다. alphaXiv에서 2,656건의 엔게이지먼트로 당일 최고 관심 논문을 기록했다. [alphaXiv](https://www.alphaxiv.org/)

### In-Place Test-Time Training: MLP 블록 재활용으로 LLM 동적 적응 — 추가 연산 무시 가능 수준

4월 7일 arXiv에 등재된 'In-Place Test-Time Training'(Guhao Feng, Shengjie Luo, Kai Hua 외)은 추론 시점에 LLM을 동적으로 적응시키는 프레임워크다. 별도의 파라미터 추가 없이 기존 MLP 블록을 재활용해 새로운 정보를 모델 내부에 통합하며, '무시할 수 있는 수준의 추가 연산'만을 요구한다고 저자들은 주장한다. 정적 웨이트로 고정된 LLM이 새로운 도메인 정보를 실시간 반영하지 못하는 한계를 훈련 비용 없이 극복하는 방향을 제시한다. alphaXiv에서 1,343건 방문을 기록했다. [alphaXiv](https://www.alphaxiv.org/)

### RAGEN-2: 에이전트 RL 추론 붕괴 탐지 — 'Template Collapse'와 SNR-Aware Filtering

4월 7일 arXiv에 등재된 RAGEN-2(Zihan Wang, Chi Gui, Xing Jin)는 강화학습으로 훈련된 AI 에이전트에서 나타나는 'Template Collapse' 현상을 최초로 정량화한 논문이다. 에이전트의 추론 경로가 표면적으로 다양해 보이지만 실제로는 입력에 무관하게 동일 패턴을 반복하는 이 현상을 SNR(Signal-to-Noise Ratio) 기반 지표로 측정하고, SNR-Aware Filtering으로 훈련 데이터 품질을 선별하는 방법을 제안한다. 에이전트 RL 훈련이 확산되는 시점에 '학습이 실제로 진행되고 있는가'를 검증하는 진단 도구를 제공한다. [alphaXiv](https://www.alphaxiv.org/)
