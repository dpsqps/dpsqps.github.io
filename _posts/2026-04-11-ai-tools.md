## 🛠️ Claude Managed Agents — 엔터프라이즈 AI 에이전트 프로덕션 인프라의 상업화

4월 8일 Anthropic이 Claude Managed Agents를 퍼블릭 베타로 출시했다. AI 에이전트를 실제 비즈니스 환경에 배포할 때 필요한 인프라 전반을 Anthropic이 관리형 클라우드 서비스로 제공하는 새로운 제품이다.

### Claude Managed Agents 퍼블릭 베타: 세션당 $0.08, Notion·Rakuten·Sentry 초기 도입

4월 8일 Anthropic이 출시한 Claude Managed Agents는 AI 에이전트의 엔터프라이즈 배포에서 가장 복잡한 요소들을 클라우드 서비스로 추상화한다. 상태 관리(state management): 에이전트가 작업 중 유지·참조해야 하는 데이터의 저장·일관성 보장을 Anthropic이 처리한다. 툴 오케스트레이션: 수신된 프롬프트에 어떤 도구를 어떤 순서로 사용할지 자동 결정. 샌드박싱·권한 제어·오류 복구·호스팅·스케일링·모니터링까지 전부 포함한다. 가격은 세션당 $0.08에 표준 Claude API 토큰 비용을 추가하는 구조다. Claude Console, Claude Code, 새 CLI를 통해 빌드·배포가 가능하다. 초기 도입 고객사: Notion(팀원이 Claude에게 할 일 직접 위임), Rakuten(Slack·Teams 연동 영업·마케팅·재무 에이전트 구축), Asana(프로젝트 관리 자동화), Sentry(버그 디버깅 에이전트와 패치 PR 자동 생성 에이전트 연동). Anthropic의 Managed Agents 출시가 산업에 주는 신호는 분명하다: 에이전트 인프라 구축이 각 기업의 자체 역량에서 외부 서비스로 이전되면서, AI 에이전트 배포 속도가 획기적으로 낮아진다.
