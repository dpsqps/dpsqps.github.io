# AI 모델 동향 — 맞춤형·소형·에이전트 특화 모델의 시대

오늘 AI 모델 분야에서는 엔터프라이즈 맞춤형, 소형 고효율, 에이전트 특화라는 세 가지 흐름이 동시에 나타났다.

## Mistral, 기업 맞춤형 AI 플랫폼 'Forge' 발표

Mistral AI가 NVIDIA GTC 2026에서 기업용 AI 플랫폼 **Mistral Forge**를 공개했다. 기존 파인튜닝이나 RAG 기반 접근 방식 대신, 기업이 자체 데이터로 AI 모델을 처음부터 직접 학습할 수 있도록 지원한다. 데이터 주권과 완전한 커스터마이징을 원하는 기업 수요를 정조준한 이 플랫폼은 OpenAI·Anthropic에 대한 정면 도전으로 평가된다. [TechCrunch](https://techcrunch.com/2026/03/17/mistral-forge-nvidia-gtc-build-your-own-ai-enterprise/)

## 문샷 AI, 트랜스포머 아키텍처 혁신 'AttnRes' 공개

중국 문샷 AI 연구진이 기존 트랜스포머의 핵심인 잔차 연결을 대체하는 **AttnRes(주의 잔차, Attention Residuals)** 기술을 발표했다. 깊은 신경망에서 발생하는 정보 소실과 기울기 소멸 문제를 구조 자체에서 해결해 LLM의 근본적인 성능 향상을 목표로 한다. 단순 알고리즘 개선을 넘어 모델 구조를 재설계하려는 시도가 주목된다. [AI타임스](https://www.aitimes.com/news/articleView.html?idxno=207982)

## Zhipu AI, 에이전트 특화 'GLM-5-Turbo' 출시

중국 Zhipu AI가 GLM-5 기반의 **GLM-5-Turbo**를 공개했다. 속도·안정성·도구 활용 능력을 강화한 에이전트 실행 최적화 모델로, 장기 작업 자동화와 복잡한 멀티스텝 과제 수행에 특화되어 있다. LLM 경쟁이 단순 성능을 넘어 실제 업무 수행 능력으로 이동하고 있음을 보여준다. [AI타임스](https://www.aitimes.com/news/articleView.html?idxno=207987)

## NVIDIA, 소형 하이브리드 모델 'Nemotron 3 Nano 4B' 공개

NVIDIA가 HuggingFace를 통해 **Nemotron 3 Nano 4B**를 공개했다. 로컬 환경에서 효율적으로 작동하도록 설계된 컴팩트 하이브리드 모델로, 엣지 디바이스와 개인 컴퓨팅 환경에서의 AI 활용 가능성을 높인다. [HuggingFace Blog](https://huggingface.co/blog/nvidia/nemotron-3-nano-4b)

## Holotron-12B, 컴퓨터 사용 에이전트 공개

HCompany가 HuggingFace를 통해 **Holotron-12B** 컴퓨터 사용 에이전트를 출시했다. GUI 환경에서 직접 업무를 수행하는 고처리량 모델로, 소프트웨어 자동화와 에이전틱 AI의 새로운 가능성을 제시한다. [HuggingFace Blog](https://huggingface.co/blog/Hcompany/holotron-12b)
