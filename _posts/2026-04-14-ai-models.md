## 🤖 Meta Muse Spark·GLM-5.1 — 독점과 오픈소스, 프론티어 공방전

이번 주 가장 주목할 AI 모델 뉴스는 두 극단에서 동시에 나왔다. Meta는 오픈소스 전략을 포기하고 독점 프론티어 모델을 출시했고, Z.ai는 오픈소스로 소프트웨어 엔지니어링 벤치마크 정상에 올랐다.

### Meta Muse Spark: 오픈웨이트 포기·HealthBench Hard 1위(42.8점)·Intelligence Index 52점

4월 8일 Meta AI Superintelligence Labs가 Muse Spark를 공개했다. 전작 Llama 4 Maverick(Intelligence Index 18점)에서 52점으로 도약하며 Gemini 3.1 Pro·GPT-5.4(모두 57점) 바로 아래 프론티어 그룹에 진입했다. HealthBench Hard에서 42.8점으로 전체 1위를 기록(GPT-5.4: 40.1점, Gemini 3.1 Pro: 20.6점)하며 의료·헬스케어 도메인 강점을 입증했다. 시각 추론 MMMU-Pro에서는 80.5%로 전체 2위(Gemini 3.1 Pro Preview 82.4% 다음). 벤치마킹에 사용한 출력 토큰이 58M으로 Claude Opus 4.6(157M)의 3분의 1 수준으로 효율적이다. 이 모델이 중요한 이유는 단순 성능이 아니라 **전략 전환** 때문이다: Meta는 약 1년간 유지하던 오픈웨이트 정책을 공식 포기하고, WhatsApp·Instagram 20억 이상 사용자에게 프론티어급 AI를 직접 제공하는 소비자 AI 플랫폼 전략으로 선회했다. [Meta AI 공식](https://ai.meta.com/blog/introducing-muse-spark-msl/) / [Artificial Analysis](https://artificialanalysis.ai/articles/muse-spark-everything-you-need-to-know)

### Z.ai GLM-5.1: MIT 라이선스 오픈소스, SWE-Bench Pro 58.4% — GPT-5.4·Claude Opus 4.6 동시 추월

4월 7일 Z.ai(구 Zhipu AI)가 GLM-5.1을 MIT 라이선스로 공개했다. 744B 파라미터 Mixture-of-Experts(활성 40B/토큰) 아키텍처로 SWE-Bench Pro에서 58.4%를 달성, GPT-5.4(57.7%)·Claude Opus 4.6(57.3%)·Gemini 3.1 Pro(54.2%)를 모두 앞질렀다. 오픈소스 모델이 전문 수준 소프트웨어 엔지니어링 벤치마크에서 독점 최강 모델 전체를 추월한 최초 사례다. 컨텍스트 윈도우 200,000 토큰, 최대 출력 131,072 토큰. API 가격 $1~$3.20/백만 토큰은 Claude Opus 4.6($15~$75) 대비 5~20배 저렴하다. Huawei Ascend 칩과 NVIDIA GPU 모두에서 실행 가능한 것도 미국 칩 제재 상황에서 주목할 특징이다. [Z.ai/Winbuzzer](https://winbuzzer.com/2026/04/09/z-ai-releases-glm-5-1-754b-model-tops-swe-bench-pro-xcxwbn/) / [GLM-5.1 Developer Guide](https://lushbinary.com/blog/glm-5-1-developer-guide-long-horizon-agentic-coding/)
