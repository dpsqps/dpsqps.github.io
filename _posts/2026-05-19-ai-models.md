## 빠른 모델과 거대 컨텍스트 — 같은 날 갈라진 두 전략

오늘 AI 모델 분야에서는 두 거대 기업이 정반대의 카드를 꺼냈습니다. 구글은 '작고 빠른' 경량 모델을 에이전트 시대의 기본 인터페이스로 전면에 내세웠고, 알리바바는 100만 토큰 컨텍스트를 갖춘 '크고 깊게 생각하는' 플래그십 추론 모델로 프런티어 경쟁에 합류했습니다.

### 구글, 'Gemini 3.5 Flash' 정식 출시 — 작고 빠른 모델을 전면에

구글이 I/O 2026 개발자 행사에 맞춰 경량·고속 모델 'Gemini 3.5 Flash'를 정식 출시했습니다. 이 모델은 Gemini API와 AI Studio는 물론, 코딩 환경인 Antigravity와 Android Studio를 통해서도 제공됩니다. 주목할 점은 이 경량 모델이 상위 모델인 Gemini 3.1 Pro를 코딩·에이전트 벤치마크에서 앞섰다는 사실입니다.

성능 수치가 이를 뒷받침합니다. Gemini 3.5 Flash는 터미널 작업 능력을 측정하는 Terminal-Bench 2.1에서 76.2%, 경제 가치 평가 지표인 GDPval-AA에서 1656 Elo, 도구 연동 능력을 보는 MCP Atlas에서 83.6%를 기록했습니다. 플래그십이 아닌 경량 모델이 이 정도 점수를 낸다는 것은 의미가 큽니다.

이 출시가 중요한 이유는 단순한 모델 업데이트를 넘어선 전략적 신호이기 때문입니다. 구글이 가장 크고 비싼 Pro 모델이 아니라, 작고 빠르고 저렴한 Flash 모델을 '에이전트 시대의 기본 인터페이스'로 전면에 내세웠다는 점입니다. 수많은 호출이 반복되는 에이전트 워크플로에서는 한 번의 최고 성능보다 빠르고 저렴한 반복 실행이 더 중요해지고 있음을 보여줍니다. [Google Blog](https://blog.google/innovation-and-ai/technology/ai/google-io-2026-all-our-announcements/)

### 알리바바, 'Qwen 3.7-Max' API를 DashScope에 공개 — 1M 컨텍스트로 프런티어 합류

알리바바가 플래그십 추론 모델 'Qwen 3.7-Max'의 API 접근을 5월 19일 DashScope에 먼저 개방했습니다. 이 모델은 5월 20일 알리바바 클라우드 서밋에서 공식 발표될 예정이었으나, API 접근만 하루 앞서 열린 것입니다. 100만(1M) 토큰 규모의 컨텍스트 윈도우와 네이티브 확장 사고(extended-thinking) 모드를 갖춘 클로즈드웨이트(closed-weight) 모델입니다.

성능 면에서도 상위권을 입증했습니다. Artificial Analysis 지능지수에서 56.6점을 받아 전체 5위에 올랐고, 소프트웨어 엔지니어링 벤치마크 SWE-Pro에서 60.6, 대학원 수준 과학 추론을 보는 GPQA Diamond에서 92.4를 기록했습니다. 가격은 입력 100만 토큰당 $2.50, 출력 100만 토큰당 $7.50로 책정됐습니다.

이 공개가 중요한 이유는 중국 AI 랩이 글로벌 상위권 수준의 추론 모델을 1M 컨텍스트와 함께 내놓으며 프런티어 경쟁에 본격 합류했음을 보여주기 때문입니다. 긴 문서 전체를 한 번에 처리하는 거대 컨텍스트와 깊은 추론 능력을 동시에 갖춘 모델이 미국 빅테크의 전유물이 아님을 확인시킨 사례입니다.
