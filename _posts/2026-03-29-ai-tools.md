## AWS 서버리스 에이전트·NVIDIA Agent Toolkit — 에이전트 도구의 확산

에이전트 AI 도구가 클라우드에서 엔터프라이즈까지 빠르게 확산되고 있습니다.

### AWS 서버리스 에이전트 플러그인 출시

Amazon Web Services가 Agent Plugin for Serverless를 출시했습니다. 서버 관리 없이 AI 에이전트를 배포·운영할 수 있어 개발 접근성이 크게 향상됩니다. 동시에 AI/ML 장학금 프로그램도 확대하여 차세대 개발자 양성에 투자하고 있습니다. [AWS Blog](https://aws.amazon.com/blogs)

### NVIDIA Agent Toolkit: 오픈소스 OpenShell 공개

NVIDIA가 GTC에서 발표한 Agent Toolkit이 구체화됐습니다. 오픈소스 OpenShell 런타임으로 자기 진화형 엔터프라이즈 AI 에이전트를 구축할 수 있으며, Adobe, Atlassian, Salesforce, SAP, ServiceNow, Siemens, CrowdStrike 등이 도입을 발표했습니다. [NVIDIA Newsroom](https://nvidianews.nvidia.com/news/ai-agents)

### OpenAI Sora 종료의 도구 시장 영향

Sora 서비스 종료로 AI 비디오 도구 시장이 재편되고 있습니다. 일일 추론 비용 ~$1,500만 대비 총 인앱 수익 $210만이라는 경제적 실패가 공개됐으며, 경쟁 비디오 도구(Runway, Pika Labs 등)가 반사이익을 누리고 있습니다. [TechCrunch](https://techcrunch.com/2026/03/29/why-openai-really-shut-down-sora/)
