## 📄 Claude Mythos + Project Glasswing — AI가 17년·27년 된 제로데이 수천 개를 자율 발견

이번 주 AI 연구 분야에서 가장 충격적인 결과는 Anthropic의 Claude Mythos가 보안 연구에서 보여준 능력이다. 단순한 취약점 발견 도구를 넘어, AI가 기존 보안 전문가의 평생 성과를 수 주 안에 달성하는 단계에 진입했음을 보여준다.

### arXiv-연계 Project Glasswing: Mythos가 Firefox 취약점 181개 익스플로잇, 전작 Opus 4.6은 2개 — 27년 된 OS 버그 포함

4월 7일 Anthropic이 공개한 Claude Mythos 프리뷰와 Project Glasswing 실험 결과가 사이버보안 커뮤니티에 충격을 주었다. 핵심 수치: Claude Mythos는 Firefox 취약점을 대상으로 한 테스트에서 작동하는 익스플로잇을 **181회** 자율 개발했다. 동일 테스트에서 전작 Claude Opus 4.6은 2회에 그쳤다. 발견된 취약점들: ① OpenBSD — 27년간 존재했던 TCP SACK 버그로 커널 크래시 유발 가능, ② Linux — KASLR 우회 기반 레이스 컨디션 권한 상승 익스플로잇, ③ FreeBSD — NFS 서버 원격 코드 실행 취약점(CVE-2026-4747, 17년 잠복). 보안 연구자 Nicholas Carlini의 평가: "지난 몇 주 동안 내 평생 찾은 것보다 더 많은 버그를 발견했다." Anthropic은 이 능력의 악용 방지를 위해 AWS·Apple·Microsoft·Google·Cisco·CrowdStrike·Nvidia·JPMorganChase 등 50개 기관에만 제한 접근하는 Project Glasswing을 운영하며, $1억 사용 크레딧과 $400만 오픈소스 보안 기부를 제공했다. 이 연구가 의미하는 것: AI가 취약점 발견에서 인간 최고 전문가를 90배 이상 능가하는 첫 번째 실증적 증거다.
