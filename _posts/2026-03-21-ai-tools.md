# 2026년 3월 21일 AI 도구 동향

## Mistral, Voxtral TTS 출시 --- 경량 음성 AI의 새 기준

Mistral AI가 경량 텍스트-투-스피치(TTS) 모델 Voxtral TTS를 발표했다. 40억(4B) 파라미터 규모로, 9개 언어를 지원하며 3초짜리 음성 샘플만으로 맞춤 음성을 생성할 수 있다. 엔터프라이즈 음성 에이전트 구축에 최적화된 이 모델은 기존 대형 TTS 모델 대비 훨씬 적은 리소스로 고품질 음성을 제공한다.

## Google Gemini Code Assist, 개인 개발자에게 무료 제공

Google은 Gemini Code Assist를 개인 개발자에게 무료로 제공하기 시작했다. 이는 GitHub Copilot과의 직접적인 경쟁을 본격화하는 움직임이다. AI 코딩 도구 시장에서 무료 접근성 확대는 개발자 생태계의 판도를 바꿀 수 있는 중요한 전환점이다.

## 백악관 AI 정책 프레임워크와 도구 생태계 영향

백악관이 발표한 국가 AI 정책 프레임워크는 '혁신 우선(Innovation-first)' 기조를 명확히 하면서 AI 도구 개발 생태계에 긍정적 신호를 보냈다. 연방 정부의 가벼운 규제 접근은 AI 도구 스타트업들의 성장에 유리한 환경을 조성할 것으로 기대된다.
