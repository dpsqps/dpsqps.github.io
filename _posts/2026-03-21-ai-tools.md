## Wipro AI-DC·Gemini Code Assist 무료화 — 엔터프라이즈 AI 도구의 확산

3월 21일 엔터프라이즈 AI 도구와 개발자 도구 시장에서 주요 움직임이 있었습니다.

### Wipro AI-DC 솔루션 출시

Wipro가 엔터프라이즈 AI 도입을 가속화하는 AI-DC(AI-Data Center) 솔루션을 출시했습니다. 데이터센터 환경을 현대화하면서 AI를 대규모로 도입할 수 있는 표준화된 보안 스택을 제공합니다. 고객 경험 운영 전반의 혁신을 목표로 합니다. [Wipro Newsroom](https://www.wipro.com/newsroom/press-releases/2026/wipro-launches-wipro-ai-dc-solution-to-accelerate-enterprise-scale-ai-adoption-and-transform-customer-experience/)

### Google Gemini Code Assist, 개인 개발자에게 무료 제공

Google이 Gemini Code Assist를 개인 개발자에게 무료로 제공하기 시작했습니다. GitHub Copilot과의 직접적인 경쟁을 본격화하는 움직임으로, AI 코딩 도구 접근성의 확대가 가속화되고 있습니다. [Google AI Blog](https://blog.google/innovation-and-ai/technology/ai/google-ai-updates-march-2026/)

### AI 코딩 도구 시장: Claude Code 만족도 1위

2026년 3월 기준 AI 코딩 도구 만족도 조사에서 Claude Code가 46%로 1위를 기록했습니다. Anthropic의 Claude Sonnet 4.6이 Claude Code에서 Opus 4.5보다 59% 더 선호된다는 통계도 발표됐습니다. [Digital Applied](https://www.digitalapplied.com/blog/ai-dev-tool-power-rankings-march-2026-claude-gemini-windsurf)
